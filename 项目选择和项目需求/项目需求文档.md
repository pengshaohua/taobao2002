**一.项目目录搭建**

> 项目名称--英文或者拼音。
>
> 项目目录一定要到服务器中
>
> 1.统一项目目录
>
> dist:线上目录.放到服务器里面的程序(优化)
>
> src:前端开发目录
>
> php:后端开发目录
>
> 2.统一使用 gulp 运行项目（等项目结束的时候做节约时间）
>
> 3.项目首页采用sass编写，其他页面布局可以采用任意方式实现
>
> 4.(jquery+闭包+面向对象)和(jquery+闭包)要求学生选择一个使用就可以了( 也赞同采用模块化开发，但必须脱离jquery )
>
> 5.统一要求上传到 github 或者 gitee（码云） -- 每天至少一次
